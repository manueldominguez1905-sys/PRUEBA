# 98 · Registro de Decisiones de Arquitectura (ADR)

Esta carpeta almacena todas las decisiones importantes tomadas durante el desarrollo del proyecto.

Cada decisión deberá indicar:

- Fecha
- Problema detectado
- Alternativas estudiadas
- Decisión adoptada
- Motivo de la decisión
- Consecuencias

El objetivo es que cualquier desarrollador o IA pueda comprender por qué se tomó una decisión, incluso varios años después.
